# Quantum Random Number Generator using Qiskit

## Overview
This project demonstrates a Quantum Random Number Generator (QRNG) using Qiskit. It uses quantum superposition to generate truly random binary outcomes.

## Technologies Used
- Python
- Qiskit
- Quantum Computing

## Features
- Creates a quantum circuit
- Applies a Hadamard gate to create superposition
- Measures the qubit to generate random results
- Simulates execution using Qiskit Aer
